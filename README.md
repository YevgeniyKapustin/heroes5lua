heroes5lua — Heroes V Lua Scripting Support for VS Code
heroes5lua adds full language support for the Heroes of Might and Magic V scripting dialect (Lua‑based).

✨ Features
🟦 Syntax Highlighting
Full highlighting for:
Lua keywords
Heroes V scripting API
Trigger functions
Game objects and events
Creature, spell, artifact, and building constants
Custom DSL constructs used by the game engine

🔧 Extension Settings
This extension contributes the following settings:

| Setting | Description |
| --- | --- |
| ``heroes5lua.language`` | en/ru |
